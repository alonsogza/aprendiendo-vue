## Iniciar con VUE-Cliente (Mis Apuntes)

1.- Tener NodeJS instalado

~~~
npm install -g @vue/cli
~~~

2.-  Crear el Proyecto

~~~
vue create aprendiendo-vue
~~~

3.- Cambiar al Proyecto y levantar el sitio

~~~
cd aprendiendo-vue

npm run serve
~~~
